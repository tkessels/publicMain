Erste Schritte in publicMAIN

publicMAIN zeichnet sich durch ein schlichtes, funktionales Aussehen aus. Die Anwendung implementiert aber zahlreiche Funktionen die Ihnen eine intuitive Bedienung ermöglichen.

Besuchen Sie die publicMAIN-Webseite für weitere Informationen oder Anwendungsupdates.

Verwenden der Anwendungssteuerelemente

Wenn Sie publicMAIN öffnen, wird Ihnen als Erstes das puristische Design auffallen. Die wesentlichen Funktionen sind über die Menüleiste erreichbar. Weitere spezifische Anwendungssteuerelemente befinden sich im Kontextmenü der Gruppen- oder Benutzerliste oder direkt im Anwendungsfenster. Neben diesen Möglichkeiten steht Ihnen eine Befehlszeile im Anwendungsfenster zur Verfügung, nähere Informationen finden Sie im Kapitel Arbeiten mit der Befehlszeile.

Sicherung Ihrer Einstellungen und Daten

Sie können optional Ihre Einstellungen und Daten über das Menü „History“ auf einen Server ablegen. Im Menü „pMAIN“ können Sie die Einstellungen zu dem Server verwalten.

Arbeiten mit der Befehlszeile

Sie können auf alle Steuerelemente der Anwendung zugreifen, indem Sie in der Texteingabeleiste folgende Befehle eingeben. Eine Kurzreferenz zu den Befehlen erhalten Sie, indem Sie „/help“ im Anwendungsfenster eingeben.

Versenden von Nachrichten in Gruppen

Sie können Nachrichten in Gruppen versenden, indem Sie die Gruppe in der Gruppenliste auswählen und im Kontextmenü „Join Group“ klicken. Im Anwendungsfenster wird eine neue Registerkarte geöffnet, dort können Sie Nachrichten an diese Gruppe versenden. Weitere Informationen finden Sie auch im Kapitel Arbeiten mit der Befehlszeile.

Versenden von privaten Nachrichten

Sie können private Nachrichten an bestimmte Benutzer versenden, indem Sie den Benutzer in der Benutzerliste auswählen und im Kontextmenü „Send private Message“ klicken. Im Anwendungsfenster wird eine neue Registerkarte geöffnet, dort können Sie Nachrichten an den ausgewählten Benutzer versenden. Weitere Informationen finden Sie auch im Kapitel Arbeiten mit der Befehlszeile.

Versenden von Dateien

Dateien können Sie an bestimmte Benutzer versenden, indem Sie den Benutzer in der Benutzerliste auswählen und im Kontextmenü „Send File“ klicken. Im Anwendungsfenster wird ein neuer Auswahldialog geöffnet, dort können Sie die Datei an den ausgewählten Benutzer versenden.

Nachrichten bestimmter Benutzer ignorieren

Die Anzeige von Nachrichten bestimmter Benutzer können Sie unterdrücken, indem Sie den Benutzer in der Benutzerliste auswählen und dort im Kontextmenü „ignore“ klicken. Im Anwendungsfenster erhalten Sie eine Bestätigung durch die Anwendung. Wenn Sie diesen Vorgang zu einem bestimmten Benutzer wieder rückgängig machen wollen, navigieren Sie in der Benutzerliste zu dem Benutzer und wählen dort im Kontextmenü „unignore“. Im Anwendungsfenster erhalten Sie eine Bestätigung durch die Anwendung. Weitere Informationen finden Sie auch im Kapitel Arbeiten mit der Befehlszeile.

Weitere Informationen zu bestimmten Benutzer anzeigen

Weitere Informationen zu einem Benutzer erhalten Sie, indem Sie in der Benutzerliste den entsprechenden Benutzer auswählen und dort im Kontextmenü „infos about“ klicken. Im Anwendungsfenster werden die verfügbaren Informationen dargestellt.Weitere Informationen finden Sie auch im Kapitel Arbeiten mit der Befehlszeile.

Benachrichtigungen

Für Informationen über eingehende Nachrichten im Benachrichtigungsfeld wählen Sie im Benachrichtigungsfeld auf dem publicMAIN Programmsymbol im Kontextmenü „Notify“ und dann den Nachrichtentyp bei dem Sie informiert werden möchten.
